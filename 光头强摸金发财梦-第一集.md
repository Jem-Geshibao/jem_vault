## Clip 1（6秒）— 分镜1：得意

```
3D动画风格，熊出没角色风格。光头强（秃顶、大圆鼻子、蓝色安全帽、橙色工装背心的瘦高中年男子）双手举着一个鼓鼓的大背包，背包缝隙透出红色光芒。他仰头大笑，表情得意忘形，身体后仰。场景为色彩鲜艳的游戏世界草地，远处有金字塔轮廓。中景，微仰拍强调得意感，镜头缓慢推近至近景。明亮阳光，暖色调，卡通渲染质感。音效：得意的笑声。对白（中文配音）：光头强"嘿嘿嘿，还得是我光头强！这都能捡到大红！"
```

**导演意图**：建立光头强的得意状态，为后续反转做铺垫。仰拍强化他的自大感。

## Clip 2（8秒）— 分镜2：丢失与震惊

```
3D动画风格，熊出没角色风格。熊二（圆脸棕熊、浅色腹部、表情夸张）刚把熊大（壮硕棕熊、稳重表情、浅色口鼻）从土坑边拉上来。熊大拍灰擦汗，松了口气。镜头从熊大向右平移到空地——原本放背包的位置空空如也，地面上出现跳动的虚线轮廓勾勒出背包形状（卡通"东西消失"效果，虚线框上下跳动两下后消散）。硬切到熊二特写：眼睛瞪圆如铜铃、嘴巴大张、全身僵硬的极度震惊表情，参考日漫夸张震惊风格。场景为土坑边的草地，游戏世界。明亮自然光。音效：熊大松口气的叹息，然后熊二惊恐的尖叫。对白：熊大"唉！终于出来了"；熊二"我的大红！！！"
```

**导演意图**：喜剧核心——从放松到震惊的剧烈反差。虚线框效果是关键笑点。

> ⚠️ **特效提示**：虚线框跳动效果（dashed bouncing outline）AI 视频可能无法完美生成。建议方案：(1) 生成时描述为"地面残留背包形状的虚线轮廓跳动消散"，若效果不佳则 (2) 后期用 AE/剪映叠加虚线框动画图层。

## Clip 3（6秒）— 分镜3：追踪

```
3D动画风格，熊出没角色风格。熊大蹲在地上仔细看着清晰的人字形鞋印，用手指着脚印延伸的方向。熊二在旁边握拳、表情愤怒。两人起身，沿着脚印方向快速跑去。镜头从过肩俯视脚印（俯拍），切换到侧面跟拍两人奔跑（平移跟拍）。草地游戏场景，远处金字塔比之前更近。明亮阳光，色彩鲜艳。音效：急促脚步声。对白：熊大"别急，好像是光头强来过了！这边有他的脚印！"；熊二"这个臭光头！走！我们去找他！"
```

**导演意图**：节奏过渡，从静态发现到动态追逐，金字塔逐渐接近暗示空间推进。

## Clip 4（7秒）— 分镜4a：发现与追逐

```
3D动画风格，熊出没角色风格。熊二扒开草丛，看见远处光头强背着鼓鼓的红色背包在奔跑。光头强边跑边回头喊话，表情紧张又得意。他跑到巨大的金字塔底部，抓住滑索，快速滑升到塔顶。镜头从草丛缝隙视角（前景虚化的草丛），切换到跟踪追逐镜头（侧面跟拍），最后仰拍跟随滑索上升至塔顶。草地到金字塔场景，金字塔有滑索设施。明亮阳光。音效：奔跑声、滑索摩擦声、风声。对白：熊二"快看！他就在那边！"；光头强"怎么又是你们！不准抢我的大红！"
```

**导演意图**：追逐高潮的第一部分——发现目标→追逐→目标逃脱升空。三个机位变化制造紧张感。

## Clip 5（7秒）— 分镜4b：逃跑与无奈

```
3D动画风格，熊出没角色风格。光头强站在金字塔塔顶，回头大笑，然后抱着背包从金字塔斜面滑下，快速消失在画面底部。切换到熊大和熊二站在金字塔塔顶，看着光头强消失的方向，两人面面相觑、一脸无奈和茫然。镜头从光头强塔顶大笑的全景（仰拍），切换到两熊面面相觑的中景（正面平拍），最后微微推近至两熊呆滞的表情特写。金字塔塔顶场景，远处游戏世界风景。明亮阳光。音效：光头强远去的笑声渐弱，然后安静。对白：光头强"哈哈哈，这下你们追不上我了！"
```

**导演意图**：喜剧收尾——光头强成功逃脱的得意 vs 二熊的无奈呆滞，形成最终的反差笑点。

## 后期制作提示

1. **对白配音**：Seedance 生成的视频可能无法准确生成中文对白口型和语音，建议所有对白后期配音添加。
2. **虚线框特效**：若 AI 生成效果不佳，使用剪映/AE 在 Clip 2 对应帧叠加虚线框动画（0.5s 出现→0.5s 跳动→0.5s 消散）。
3. **角色一致性**：5个 clip 之间角色外观可能存在偏差，建议使用相同角色描述词，必要时通过 R2V（参考图生视频）模式固定角色形象。
4. **转场**：Clip 2 内部有一个"硬切"（平移→震惊特写），可在生成后剪辑时处理；其余 clip 之间用"硬切"或"快切"衔接，保持动画节奏感。
5. **音效层**：建议后期统一添加脚步声、滑索声、风声等环境音效，增强追逐感。
