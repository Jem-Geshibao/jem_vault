# jem_vault 🎬

《熊出没》（Boonie Bears）漫剧创作笔记与 AI 生成提示词库

## 📋 项目概述

本仓库用于存储《熊出没》系列漫剧的创意开发笔记和 Seedance 2.0 AI 视频生成提示词。旨在记录每一集的故事设定、角色连续性约束、镜头设计、音频方案等制作细节，为 AI 辅助创作提供精细的参考框架。

## 📁 仓库结构

```
jem_vault/
├── README.md                           # 本文件
└── 光头强摸金发财梦-第二集.md         # 第二集完整提示词与制作笔记
```

## 📖 内容说明

### 第二集：《光头强摸金发财梦 第2集 — Seedance 2.0 提示词（T2V 多镜头）》

**核心内容**：
- **模式**：T2V（文本到视频生成，单次生成 4 个镜头，约 30–38 秒）
- **风格**：3D CGI 国产动画喜剧，熊出没风格，明亮饱和色彩
- **故事线**：承接第一集结尾，光头强拿着红背包滑下金字塔，二熊发现背包已空而展开追逐

**文件包含**：

1. **全局连续性锁定（Continuity Locks）**  
   - 角色造型固定（光头强、熊大、熊二、红背包等）
   - 场景与道具一致性
   - 禁止事项（写实化、改变造型、真人演员等）

2. **四个镜头的详细设计**
   - **Shot 1 (0:00–0:08)**：喜剧反转·真相大白
   - **Shot 2 (0:08–0:18)**：发现丢失·喜剧震惊
   - **Shot 3 (0:18–0:27)**：推理+追击
   - **Shot 4 (0:27–0:38)**：追逐高潮·二次逃脱

3. **每个镜头包含**：
   - 镜头描述与视觉设定
   - 时间码
   - 剧情意图
   - 音频设计（音效、BGM、对话）
   - 制作约束

4. **英文精简版**  
   为国际制作团队提供的英文版本

5. **制作建议**
   - 生成策略对比（单次 vs 分次生成）
   - 特效处理建议
   - 衔接策略
   - 版权声明

## 🎯 使用指南

### 对于 AI 视频生成

1. 复制相应集数的提示词内容
2. 根据需要选择完整版或英文版本
3. 遵循"全局连续性锁定"中的约束条件
4. 参考"制作建议"选择合适的生成策略

### 对于创意开发

- 作为故事创意和角色设计的参考
- 镜头设计和视觉故事板的规范模板
- 音频设计方案的示例

## 📝 笔记特点

✅ **详细的场景描述** — 包含光线、构图、角色表情等视觉细节
